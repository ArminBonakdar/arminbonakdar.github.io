Fatigue Detection in Repetitive Manual Handling Tasks using Joint Angle Features: A Machine Learning Approach

Predicting fatigue in material handling tasks is crucial to prevent work-related injuries and enhance worker safety. Fatigue affects movement coordination, which can be identified using inertial measurement units (IMUs) found in smart devices. This study utilizes machine learning approaches to detect muscle fatigue by analyzing IMU recordings.
